## 🍎 macOS версия - ГОТОВА ✅

### 📁 Файлы:
- **`dist/SpellChecker`** - готовый исполняемый файл (~9 МБ)
- `run_spellchecker.sh` - скрипт запуска
- `demo.sh` - демонстрация

### 🚀 Использование:
```bash

./run_spellchecker.sh
```

```

```

## 🤖 Android версия - ГОТОВА ✅
